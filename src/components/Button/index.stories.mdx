## Primary Button
This is the primary button on a form/page.  Not all buttons should be primary to differentiate between primary and alternative paths for the user.

<Canvas>
    <Story name="Primary" args={{ size: 'default', text: 'Click me', disabled: false, variant: 'contained' }}>
       {({text, ...args}) => <Button {...args}>{text}</Button>}
    </Story>
</Canvas>


## Secondary Button
The secondary button is useful for having additional options outside of the normal flow
<Canvas>
    <Story name="Secondary" args={{ size: 'default', text: 'Click me', disabled: false, variant: 'outlined' }}>
        {({text, ...args}) => <Button {...args}>{text}</Button>}
    </Story>
</Canvas>


## Icons
The buttons also support icons with the startIcon and endIcon properties.  Here is an example.

        {({text, ...args}) => <Button endIcon={Icon} {...args}>{text}</Button>}
    </Story>
</Canvas>
