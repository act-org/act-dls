## Primary Button

The primary button is used to designate and handle a primary action the user
should take on a form or page. Not all buttons on the page should be primary to
differentiate between primary and alternative paths for the user.

<Canvas>
  <Story name="Primary" args={{ children: 'Click me', color: 'primary' }}>
    {Template.bind({})}
  </Story>
</Canvas>

## Secondary Button

The secondary button is useful for having additional options outside of the normal flow

<Canvas>
  <Story
    name="Secondary"
    args={{ children: 'Click me', disabled: false, variant: 'outlined' }}
  >
    {Template.bind({})}
  </Story>
</Canvas>

## Icons

The buttons also support icons with the startIcon and endIcon properties. Here is an example.

<Canvas>
  <Story
    name="Icons"
    args={{ children: 'Click me', disabled: false, variant: 'contained' }}
  >
    {({ children, ...args }) => (
      <Button endIcon={Icon} {...args}>
        {children}
      </Button>
    )}
  </Story>
</Canvas>
